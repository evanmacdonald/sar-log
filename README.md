# SAR Log

SAR Log is a native iOS / SwiftUI app for BC search and rescue members to replace the paper notebook during a task.

The project scope, product decisions, and definition of done live in [CHARTER.md](./CHARTER.md). The implementation sequence lives in [PLAN.md](./PLAN.md).

## Development

- Open `SARLog.xcodeproj` in Xcode.
- Build the `SARLog` app target for an iOS simulator or device.
- Keep changes scoped to one item from [PLAN.md](./PLAN.md) per PR.
